# Advent of Code - Personal Solutions

This repository contains a collection of my original solutions to algorithmic challenges from the Advent of Code platform. I hope you find the provided code and the approaches taken to solve specific problems interesting.

## Problem-Solving Methodology

The process of developing these solutions generally relied on a two-step, iterative approach:

* **Initial Implementation (Brute-force):** In the first phase, the primary focus was on delivering a working solution and obtaining the correct result as quickly as possible. The initial code often utilized brute-force methods without strict consideration for time and memory constraints.
* **Performance Optimization:** After verifying the correctness of the algorithm, the code underwent refactoring. The implementation was improved by applying appropriate data structures and optimizing computational complexity. The main objective of this phase was to drastically reduce the execution time.

## Technologies and Execution

* **Programming Language: Python
* **How to Run:** To test the code for a specific day, execute the main script while providing the corresponding input data file (`input.txt`).
